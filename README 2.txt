DESCRIPCION:
En este proyecto trabajamos en La aplicación que permite el registro de nuevos libros adquiridos por el usuario, 
como también registrar detalles del autor de cada libro. Permitir la categorización 
de la colección, como también el registro de libros deseados, para compras futuras.

INSTALACION:
Para la instalacion del proyecto debemos realizar los siguientes pasos:
1. Clonar repositorio de GitHub.
Esto lo realizamos desde Visual Studio Code con el siguiente Enlace: https://github.com/fatejohann/proyectoDAW.git

2. Abrimos el repositorio clonado y verificamos los archivos.
En este paso lo que realizamos es la apertura de los archivos que estan en el repositorio para poder llevar a cabo la compilacion del proyecto.

3. Instala las Dependencias::
Ya que nuestro proyecto lo hemos desarrollado con el framework de JavaScript "Angular", necesitamos ejecutar la siguiente linea de comandos en nuestra terminal "npm install", esto nos ayudara a que nuestro proyecto quede configurado correctamente con Angular.
Para Firebase debemos ejecutar el siguiente comando: "ng add @angular/fire".
Con esto nuestro proyecto ya estaria listo para correr.

4. Compilar nuestro proyecto.
Lo que debemos hacer para poder ver funcional nuestro proyecto es ejecutar el comando "ng serve" o tambien podemos utlizar "npm start".
